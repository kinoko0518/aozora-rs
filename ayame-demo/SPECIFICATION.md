本ページは以下の仕様を満たす。それぞれの機能には最新のWASMビルドが使用され、更新はpre-commitで管理される。

# リアルタイムプレビュー
左右に分かれた入力欄とリアルタイムプレビューを備え、以下の文章がデフォルトで入力されている。
```
［＃２字下げ］有明［＃「有明」は中見出し］


起伏の雪は
あかるい桃の漿《しる》をそそがれ
青ぞらにとけのこる月は
やさしく天に咽喉《のど》を鳴らし
もいちど散乱のひかりを呑む
　　（波羅僧羯諦《ハラサムギヤテイ》　菩提《ボージユ》　薩婆訶《ソハカ》）
［＃地付き］（一九二二、四、一三）
```

# ファイル変換
UTF-8とShift_JISのうち、入力ファイルのエンコーディングを選択する切り替えボタンとファイルのアップロードボタンを備え、.zipまたは.txt形式のファイルが入力されたらダウンロードボタンとHTMLで読むボタンを表示、HTMLで読むボタンが押されたら生成されたXHTMLを開き、ダウンロードボタンが押されたらEPUBをダウンロードする。
