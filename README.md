# aozora-rs-core
テキストを中間表現に変換する責任を持っています。内部的にはトークン化層、スコープ化層、再トークン化層の三層構造になっており、エラーは自動修復されながら蓄積されます。

# aozora-rs-gaiji
※［＃と］の中身の部分（※［＃、］は入力に含めないでください）を受け取り、Unicodeから変換 → Shift_JISコードから変換 → 変換テーブルから変換の三段階を試してUTF-8化された文字を返します。
合字である可能性を踏まえ返り値がOption<String>であることに注意してください。

# aozora-rs-xhtml
aozora-rs-coreが生成した中間表現からXHTMLを生成します。
